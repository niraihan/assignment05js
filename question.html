Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML → HTML কোডসহ সম্পূর্ণ কনটেন্ট পায় বা সেট করে এটি HTML ট্যাগসহ কনটেন্ট ফেরত দেয় এবং HTML রেন্ডার করে। নতুন HTML ট্যাগ সেট করলে তা ডাইনামিকভাবে পার্স ও এক্সিকিউট হয়।

innerText → শুধু দৃশ্যমান (visible) টেক্সট পায় বা সেট করে এটি শুধু দৃশ্যমান টেক্সট রিটার্ন করে, HTML ট্যাগ উপেক্ষা করে।

textContent → সব টেক্সট পায়, Hidden এলিমেন্টসহ এটি HTML ট্যাগ বাদ দিয়ে সম্পূর্ণ টেক্সট ফেরত দেয়। Hidden (display: none;) এলিমেন্টের টেক্সটও পাওয়া যায়।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি মূল parent এলিমেন্টে ইভেন্ট লিসেনার যোগ করা, যাতে তার(child) এলিমেন্টগুলোর ইভেন্ট পরিচালনা করা যায়। এটি Event Bubbling এর উপর নির্ভর করে যেখানে একটি ইভেন্ট শুরুতে নির্দিষ্ট এলিমেন্টে ঘটে, তারপর তার প্যারেন্ট এলিমেন্টে বাবল আপ হয়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি ইভেন্ট যখন কোনো চাইল্ড এলিমেন্টে ঘটে, তখন সেটি প্যারেন্ট এলিমেন্টের দিকেও ছড়িয়ে পড়ে (bubble up করে)। এটি DOM ইভেন্ট Propagation একটি অংশ, যেখানে একটি ইভেন্ট প্রথমে যেই এলিমেন্টে ঘটে, সেখান থেকে তার প্যারেন্ট, গ্র্যান্ডপ্যারেন্ট হয়ে document পর্যন্ত ছড়িয়ে যায়।
